## ⚡ Get Started in 3 Steps

### Step 1: Install (2 minutes)
```bash
npm install
```

### Step 2: Setup Backend (5 minutes)
1. Go to [supabase.com](https://supabase.com) and create project
2. Open SQL Editor in Supabase Dashboard
3. Copy entire `supabase-setup.sql` file
4. Paste and click "Run"
5. Get your credentials from Settings → API
6. Update `.env` file

### Step 3: Run (1 minute)
```bash
npm run dev
```

**Open http://localhost:5173 and login with:**
- Email: `owner@autorepair.com`
- Password: `owner123456`

**🎊 That's it! You're running the CRM!**
