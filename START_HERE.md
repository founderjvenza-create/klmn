## 🎯 What Is This?

A modern web application for auto repair shops featuring:
- Customer and vehicle management
- Online appointment booking
- Job tracking and assignment
- Invoice generation
- Role-based access control (Owner, Receptionist, Worker, Customer)

**Tech Stack**: React + Tailwind CSS + Supabase + Vercel

---

## ⚡ Quick Start (5 Minutes)

### 1. Install Dependencies
```bash
npm install
```

### 2. Setup Supabase Backend
1. Create account at [supabase.com](https://supabase.com)
2. Create new project
3. Go to SQL Editor
4. Copy and paste **entire** `supabase-setup.sql` file
5. Click "Run"

### 3. Get Your Credentials
1. In Supabase: Settings → API
2. Copy "Project URL" and "anon public key"
3. Update `.env` file with your credentials

### 4. Run the App
```bash
npm run dev
```

### 5. Login
Open http://localhost:5173 and use test credentials from `TEST_USERS.md`

**🎉 Done! You're running the CRM!**
