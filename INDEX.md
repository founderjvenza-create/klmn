## 🚀 Setup in 3 Steps

### 1. Install (2 minutes)
```bash
npm install
```

### 2. Setup Backend (5 minutes)
- Create Supabase project
- Run `supabase-setup.sql`
- Get credentials
- Update `.env`

### 3. Run (1 minute)
```bash
npm run dev
```

**→ Open http://localhost:5173**
